# ⭐ Overview
This project was implemented to simulate a Canberra GC3018.


# 🔧 Requirements
  * Ubuntu 20.04/Window 10
  * Geant4.11.1.2
  * Root data analysis framework

# 🏃‍♂️ How to run
* Run example in interactive mode
    ```c++
    ./sim
    ```
* Run example in batch mode
    ```c++
    ./sim run.mac
    ```
* Plot results
  ```
    root plot.c
  ```
